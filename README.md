# GraphCalc

Десктопное приложение для построения графиков математических функций и поиска экстремумов методом дихотомии.

---

## Возможности

- **Простой график** — построение f(x) на заданном отрезке
- **График с параметром** — несколько функций с разными значениями параметра `a` на одном графике
- **Поиск максимума и минимума** — метод дихотомии с отображением результата на графике
- **Интерактивный холст** — zoom, pan, сброс вида через встроенный тулбар
- **Настройки** — сохраняемый диапазон, тема оформления, режим цвета графиков

---

## Поддерживаемый синтаксис

| Ввод | Интерпретация |
|---|---|
| `x^2` или `x**2` | степень |
| `2x`, `2sin(x)` | умножение |
| `\|x\|` | модуль |
| `sin`, `cos`, `tan` | тригонометрия |
| `sqrt(x)` | квадратный корень |
| `cbrt(x)` | кубический корень |
| `ln(x)`, `log(x)` | натуральный логарифм |
| `log2(x)`, `log10(x)` | логарифм с основанием |
| `exp(x)` | экспонента |
| `pi`, `e` | константы |
| `3,14` | запятая как десятичный разделитель |
| `Sin(x)`, `COS(x)` | регистр не важен |
| `a` | параметр (для вкладки "Параметр") |

## Установка и запуск

### Запуск из исходников

```bash
pip install customtkinter matplotlib numpy
python main.py
```

### Готовый .exe

Скачай `GraphCalc.exe` из папки `dist/` и запусти — Python не нужен.

Рядом с `.exe` автоматически создастся `config.json` с настройками.

## Структура проекта

```
GraphCalc/
├── main.py                  — файл для запуска самого приложения
├── config.json              — настройки пользователя (создаётся автоматически)
├── assets/
│   └── GRAPH2.ico           — иконка приложения
├── core/
│   ├── graph.py             — математика (валидация функйии, дихотомия, графики)
│   └── config.py            — загрузка и сохранение настроек
└── ui/
    ├── app.py               — главное окно, сборка вкладок
    ├── tab_graph.py         — вкладка простого графика
    ├── tab_parameter.py     — вкладка графика с параметром
    ├── tab_dichotomy.py     — вкладка дихотомии
    └── tab_settings.py      — вкладка настроек
```

## Настройки

Хранятся в `config.json` рядом с `.exe` или рядом с `main.py` при запуске из исходников:

```json
{
    "range_a": -20,
    "range_b": 20,
    "color_mode": "all_different",
    "theme": "dark",
    "window_width": 1100,
    "window_height": 680,
    "last_tab": 0
}
```

| Ключ | Описание |
|---|---|
| `range_a`, `range_b` | диапазон построения по умолчанию |
| `color_mode` | `all_different` — каждый график свой цвет, `by_parameter` — одинаковый цвет для одинакового значения `a` |
| `theme` | `dark`, `light`, `system` |
| `window_width`, `window_height` | размер окна при запуске |
| `last_tab` | индекс последней активной вкладки (0–3) |

<summary>## Информация о сборе .exe файла</summary>

**Удаление старого кеша**

```powershell
Remove-Item -Recurse -Force build
Remove-Item -Recurse -Force dist
Remove-Item -Force GraphCalc.spec
```

```bat
pyinstaller --onefile --windowed --name=GraphCalc --icon=assets/GRAPH2.ico --add-data "assets/GRAPH2.ico;." --add-data "core;core" --add-data "ui;ui" --collect-all customtkinter --collect-all matplotlib main.py
```
